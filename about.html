  • What special features have you done, or attempted, in creating the site that we should know about?

    • I have attempted to use Bootstrap 4 to create this website, along with a service called FontAwesome.

  • Which parts did you have trouble with?

    • I had particular trouble with writing to/reading from the database through PHP from my home computer. It was only after I put my code on cmslamp that I realized it worked and it was just that I couldnt access the datase through code from my home computer, regardless of RASS ticket being open. I also had trouble with the mobile collapse menu when the screen size is less than 768pixels. I couldnt figure out why the Bootstrap wasn't working, but after some Googling, i found that the issue was I needed to makesure Bootstraps j-query script was called before the Bootstrap CSS script.

  • What would you like to do better next time?

    • I feel as though some of the PHP code is a little convaluted because it was written while still learning how to do certain things, and this may have lead to unneeded, messy code. I would also prefer not to use Bootstrap again as i found it quite annoying having to manually cancel what Bootstrap was doing, CSS wise, to be able to write my own CSS for a particular element.

  • What references/sources you have used to help you learn how to create your website?

    • PHP Documentation, w3Schools, Bootstrap Documentation and various YouTube videos have been very helpful with the understanding of some of the input validation processess and styling of the webpages along with the handling of MySQL queries with PHP.

  • What you have learnt along the way?

    • I have learnt a considerable amount about Bootstrap and how it works, along with how to create and query a MySQL database with PHP and to validate user input through HTML patterns.
